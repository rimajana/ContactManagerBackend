I. Contact Manager is an website where an user logs in /registers if he doesn't has an account.Then he can store all his contacts.

II. Functionality:
  1. User Login In / Register
  2. After login user can 
     2.1 Create Contact
     2.2 Get all his Contacts
     2.3 Get a particular Contact
     2.4 Update a Contact
     2.5 Delete a Contact
  3. Jwt Authentication
 
 III. This is the backend of the particular Website.
